improved-rotary-phone implementa um envio local de mensagens para um servidor usando a linguagem go 
em uma máquina LINUX:

O programa deve ser iniciado primeira mente no computador chamado de SERVIDOR e após isso, inicie
outra instancia de terminal para iniciar os clientes.

Todas as mensagens de clientes serão enviadas ao servidor o qual poderá ver com uma identificação.


Deve inicialmente importar o pacote websocket do gorrila

                :~ go get "github.com/gorilla/websocket"

Após isso é só iniciar atraves do:

                :~ go run main.go
